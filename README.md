# ⏱️ WorkClock Pro

A modern desktop employee time tracking application built with Python, Tkinter, SQLite, and Supabase.

## ✨ Features

- ✅ Employee Login
- ✅ Daily Time Tracking
- ✅ Break Timer
- ✅ Live Work Duration
- ✅ Daily Target Progress
- ✅ SQLite Local Database
- ✅ Supabase Cloud Sync
- ✅ Automatic Heartbeat
- ✅ Employee Registration
- ✅ Daily Timesheet Export
- ✅ Modern Dark UI

## 🛠 Tech Stack

- Python

- SQLite

## 🚀 Getting Started

1. Download **WorkClock.exe** from the repository or Releases.
2. Launch the application.
3. Enter your name.
4. Start tracking your work.

> No Python installation is required to run the executable.
